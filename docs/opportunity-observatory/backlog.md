# Opportunity backlog

Record reviewed opportunities here. The scheduled observatory may read this file but must not change it.

## Status definitions

- **Accepted:** Approved for a separate implementation task.
- **Clarify:** More evidence or a narrower proposal is required.
- **Deferred:** Credible, but blocked by timing, capacity, or a prerequisite.
- **Rejected:** Not useful, duplicative, unsupported, or outside scope.
- **Completed:** The accepted work was implemented; note the resulting page or artifact.
